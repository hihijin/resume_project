# 3. GIF
home 페이지와 project페이지로 구성되어있습니다.
햄버거메뉴를 클릭하면 사이드바가 나타나고 버튼을 클릭하면 각 페이지로 이동합니다.
이메일을 누르시면 메일보내기 모달창이 나타나고, 모달창에서 이메일을 작성 후 바로 저에게 보낼 수 있습니다.
채널톡으로 간단한 문의사항을 보내실 수 있습니다.
